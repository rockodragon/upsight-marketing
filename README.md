## Setup (10 minutes)

1. **Move `CLAUDE.md`** to your UpSight repo root: `~/code/upsight/CLAUDE.md`
2. **Edit `CLAUDE.md`** вЂ” confirm the vault path on line 6
3. **Move the rest** into your Obsidian vault location, e.g. `~/Obsidian/upsight-vault/`
4. **Edit `10-build/repo-index.md`** вЂ” confirm your actual repo path
5. **Schedule weekly review** as a recurring calendar block on both calendars
   (rickmoy@gmail.com + rick@deeplight.digital), Sunday evening or Monday morning, 30-45 min
6. **Open Claude Code in the repo** вЂ” it'll auto-read `CLAUDE.md` and follow the protocol

## First weekly review (do it this Sunday)

1. Open `40-ops/weekly-review.md` and run the steps top-to-bottom
2. Step 5 rewrites `00-control/priorities.md` with your real top 5 for the week
3. The starter `priorities.md` is pre-seeded with my best guess вЂ” replace with your real picks

## Daily mini-rituals

- **Morning:** open `priorities.md` (that's the day's filter)
- **End of day:** append to `00-control/status.md` + update `20-gtm/pipeline.md`

## Two-tool stack

- **Claude Code** = execution (build + GTM file ops)
- **Claude chat** = strategy + pressure-testing
- Cowork dropped вЂ” see `00-control/decisions.md`
