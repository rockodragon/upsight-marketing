Quick start

Add to Claude Desktop in 30 seconds.

claude_desktop_config.json 
{
  "mcpServers": {
    "upsight": {
      "command": "npx",
      "args": ["@upsight/mcp-server"],
      "env": {
        "UPSIGHT_API_KEY": "your-api-key"
      }
    }
  }
}
